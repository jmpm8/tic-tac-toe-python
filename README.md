# Tic-Tac-Toe (Tres en raya)

Proyecto final del curso **PCEP – Certified Entry-Level Python Programmer** (OpenEDG).

## Descripción
Juego de tres en raya por consola. El usuario juega con 'O' y la máquina con 'X'.
La máquina siempre coloca la primera 'X' en el centro y elige sus movimientos de forma aleatoria.

## Conceptos aplicados
- Listas bidimensionales (tablero como lista de listas)
- Funciones con parámetros y retorno de valores
- Bucles while y control de flujo
- Módulo random (randrange)
- Validación de entrada del usuario

## Cómo ejecutarlo
python Tic_tac_toe_(Tres_en_raya).py
